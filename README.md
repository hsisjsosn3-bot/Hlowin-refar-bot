## Deploy on Render
1. Upload all files to GitHub.
2. Create Web Service on Render, connect repo.
3. Set environment variables (BOT_TOKEN, ADMIN_IDS, etc.).
4. Deploy.

## Commands
- /start – Main menu
- /admin – Admin panel (admin only)
